# Nikita's Curry Corner - A Canada-Based Cloud Kitchen with Real-Time Order Management and Delivery Tracking

[![Nikita's Curry Corner](https://bharathatechno.com/_next/image?url=%2Fimages%2Fhome%2Fnikitas-curry-corner-case-study.webp&w=1920&q=75)](https://bharathatechno.com/work/nikitas-curry-corner)

### Technologies Used

- NextJs
- NodeJs
- ExpressJs
- MongoDB

[**Nikita's Curry Corner**](https://bharathatechno.com/work/nikitas-curry-corner)

Nikita's Curry Corner is a cloud kitchen offering authentic Indian cuisine, we developed a dynamic web application that allows users to place orders online, manage menus, and track deliveries in real-time.

### Key Features

- Online ordering system
- Menu management
- Real-time delivery tracking
